# GalacticGroups

Данное приложение находится в активной(не очень) стадии разработки 
на основе добровольного неоплачиваемого труда студента(ов) ЛЭТИ.

# Документация api

Её можно почитать по url _"/swagger-ui.html"_.

### Роли
В рамках переменных среды можно задать логин и пароль __админа__.  
__Админ__ имеет доступ к организациям - может создавать их, блокировать им доступ и тд.
Также __Админ__ может создавать аккаунты для персонала организаций, например, для главного лица организации("__Собственника__"), 
которое будет(или нет) иметь возможность создавать аккаунты для остального персонала.

Всем лицам в организации(например, "ЛЭТИ") доступны данные по студентам.
